# Jakob_Hardiman_Designs

Personal portfolio site for Jakob Hardiman — Graphic Design & Branding.

## Structure

```
├── index.html        # Main HTML
├── css/
│   └── styles.css    # All styles including responsive/mobile
├── js/
│   └── main.js       # Slideshow, clock, and mobile interactions

```
